# 同花顺复盘稳定捕获扩展

## 安装

1. 双击项目根目录里的 `启动复盘助手.command`，它会先打开同花顺投资账本，再打开唯一正式私有 Sites 复盘网站。
2. 打开 Edge，进入 `edge://extensions/`。
3. 打开“开发人员模式”。
4. 点击“加载解压缩的扩展”。
5. 选择这个文件夹：`tools/tzzb-edge-extension`。

## 使用

安装后打开或刷新同花顺投资账本页面即可自动捕获。扩展只把已完成交易日的同花顺投资账本只读响应同步到本机 `http://127.0.0.1:8787`；如果 helper 暂时未就绪，数据会先存在扩展里，稍后自动重试。扩展会自动取得本机生成的写入令牌，其他网页不能读取或伪造本机同步。helper 由独立的登录启动服务常驻运行；收盘定时任务只打开同花顺页面，不会反复打开复盘网站。正式复盘页登录后会自动读取最新已核验的资金、仓位、持仓和交易。
