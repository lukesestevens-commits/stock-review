# 同花顺复盘稳定捕获扩展

## 安装

1. 双击项目根目录里的 `启动复盘助手.command`，它会打开复盘网站和同花顺投资账本。
2. 打开 Edge，进入 `edge://extensions/`。
3. 打开“开发人员模式”。
4. 点击“加载解压缩的扩展”。
5. 选择这个文件夹：`tools/tzzb-edge-extension`。

## 使用

安装后打开或刷新同花顺投资账本页面即可自动捕获。扩展只把当天捕获到的同花顺投资账本只读响应同步到本机 `http://127.0.0.1:8787`；如果复盘助手没启动，当天数据会先存在扩展里，稍后自动重试。复盘页检测到新同步后会自动读取当天的资金、仓位、持仓和交易。
